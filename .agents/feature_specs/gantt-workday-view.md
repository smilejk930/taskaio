# Feature Spec: 간트차트 워킹데이 전용 뷰 (Skip Weekends/Holidays)

## 목적
간트차트의 타임라인(오른쪽 그리드)에서 토요일, 일요일 및 공공 휴일을 숨겨서 실제 업무가 진행되는 '워킹데이' 기준으로만 일정을 시각화하고 관리할 수 있도록 개선합니다.

## 사용자 시나리오
1. 사용자가 '일정 현황' 탭으로 이동한다.
2. 상단 툴바의 스케일 조절 버튼 우측에 있는 '워킹데이만 보기' 토글을 활성화한다.
3. 간트차트에서 토요일, 일요일(주말)이 실제 타임라인 칸(Cell)에서 사라진다.
4. **공공 휴일 및 연차 데이터는 타임라인을 차지하며 그대로 표시된다.**
5. 사용자가 토글을 비활성화하면 다시 전체 날짜(주말 포함)가 표시된다.

## DB 변경사항
- 없음 (기존 `holidays` 및 `tasks` 데이터 활용)

## 구현 목록
### Pages / Client Views
- `ProjectClientView.tsx`: 
  - `showWorkdaysOnly` 로컬 상태 추가
  - 간트 탭 컨트롤바에 '워킹데이만 보기' Switch 또는 Toggle UI 추가
  - `GanttChart` 컴포넌트에 해당 프롭 전달

### Components
- `GanttChart.tsx`:
  - `showWorkdaysOnly: boolean` 프롭 지원
  - `gantt.ignore_time` 설정을 통한 타임라인 필터링:
    - `date.getDay() === 0 || date.getDay() === 6` (주말) 만 필터링
  - 필터 변경 시 `gantt.render()` 호출로 UI 즉시 갱신

### 재사용
- `GanttChart.tsx` 내의 기존 `holidayInfoMap` 및 `holidayTooltip` 로직 연동

## 엣지 케이스 처리
- **일정 드래그**: 주말이 숨겨진 상태에서 일정을 드래그할 때 숨겨진 그리드 위로 마우스가 가더라도 정상적인 워킹데이에 스냅되도록 설정.
- **기간(Duration) 계산**: 주말이 숨겨졌을 때 업무 바의 시각적 길이는 줄어들 수 있으나, 데이터상의 `duration` 값 정합성 유지 확인.

## 완료 기준
- [ ] '워킹데이만 보기' 토글 스위치 사용자 노출
- [ ] 토글 켜기: 타임라인에서 토/일 및 공공 휴일 섹션 제거
- [ ] 토글 끄기: 전체 날짜 복구
- [ ] 스케일(일/주/월) 변경 시에도 워킹데이 필터 설정이 유효하게 유지됨
