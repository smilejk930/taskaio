# [Feature Spec] 업무 상세 다이얼로그 필드 높이 조절 기능

## 개요
업무 상세 정보 관리 다이얼로그에서 업무명과 업무 설명 필드를 마우스 드래그로 자유롭게 높이 조절 가능하도록 변경합니다.

## 변경 사항
1. **업무명 (title) 필드 변경**
   - `<Input>` 컴포넌트를 `<Textarea>` 컴포넌트로 변경.
   - `resize-y` 클래스를 추가하여 세로 방향 조절 허용.
2. **업무 설명 (description) 필드 변경**
   - `<Textarea>` 컴포넌트에서 `resize-none` 클래스 제거.
   - `resize-y` 클래스를 추가하여 세로 방향 조절 허용.

## 기술적 상세
- 기존의 `Input` 컴포넌트는 단일 행만 지원하므로, 여러 줄 입력과 높이 조절을 위해 `Textarea`로 교체합니다.
- 조절 핸들이 기본적으로 노출되도록 별도의 라이브러리 사용 없이 브라우저 기본 기능을 활용합니다.

## 기대 효과
- 긴 업무명이나 자세한 업무 설명을 입력할 때 입력창의 크기를 유동적으로 조절하여 작업 편의성 증대.
