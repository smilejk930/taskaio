# Walkthrough: 간트차트 휴일 헤더 툴팁 기능 구현

일정 현황(간트차트)의 상단 날짜 헤더(`day scale`)에 마우스 오버 시 등록된 휴일 명칭을 표시하는 기능을 구현하였습니다.

## 1. 구현 내용

### 1.1 데이터 매핑 고도화 (`src/components/gantt/GanttChart.tsx`)
- 기존에 휴일 타입(`public`/`leave`)만 관리하던 `holidayDateMap`을 `holidayInfoMap`으로 업그레이드하였습니다.
- 한 날짜에 여러 휴일이 겹칠 수 있는 케이스를 고려하여 `names: string[]` 배열 필드를 추가하고, 중복 없이 휴일 명칭을 합산하도록 로직을 수정하였습니다.

### 1.2 간트 스케일 헤더 템플릿 수정
- `gantt.config.scales` 설정의 `day` 단위 `format` 속성에서 `holidayInfoMap`을 참조하도록 수정하였습니다.
- 해당 날짜에 휴일 정보가 있으면 브라우저 기본 툴팁인 `title` 속성이 포함된 HTML을 반환합니다.
    - 예: `<span title="추석, 개인연차">30 (월)</span>`

### 1.3 렌더링 최적화
- `useMemo`와 `useRef`를 사용하여 데이터 가공 작업이 과도하게 발생하지 않도록 성능을 고려하였습니다.
- 휴일 데이터가 변경될 때만 간트차트가 새로 고쳐지도록 종속성을 업데이트하였습니다.
