# Walkthrough: 간트차트 주말 숨김 (상시 적용)

간트차트의 타임라인(오른쪽 그리드)에서 토요일과 일요일을 상시 숨기고, 업무 기간을 워킹데이(월-금) 기준으로만 계산하도록 개선했습니다.

## 구현된 파일
- **수정**: [GanttChart.tsx](file:///d:/develop/workspace/taskaio/src/components/gantt/GanttChart.tsx)
  - `gantt.config.work_time = true` 설정을 통해 주말을 제외한 실제 작업 일수(duration)가 계산되도록 변경했습니다.
  - `gantt.ignore_time` 로직을 추가하여 토요일(6)과 일요일(0)을 타임라인 렌더링에서 제외했습니다.
  - 타입스크립트 린트 에러를 방지하기 위해 `ganttInstance`에 대한 타입 캐스팅을 적용하고 ASI 모호성(세미콜론 누락)을 해결했습니다.

## 완료 기준 체크
- [x] 간트차트 타임라인에서 토요일, 일요일 날짜가 보이지 않음
- [x] 업무 바의 길이가 주말을 건너뛰고 워킹데이 칸에만 맞춰짐
- [x] 별도의 토글 버튼 없이 상시 적용됨
