# Walkthrough: Gantt Holiday Tooltip Improvement

간트차트 일자 헤더의 휴일 정보 툴팁 반응 속도 및 디자인을 개선했습니다.

## 변경 내용

### 1. 브라우저 기본 툴팁 제거 및 커스텀 마크업 도입
- 기존 `title` 속성을 사용하던 방식에서 `.holiday-scale-label` 클래스와 `data-holiday-names` 속성을 가진 `div`로 변경했습니다.
- 이를 통해 브라우저 특유의 툴팁 표시 지연(약 1초)을 제거하고 즉각적인 반응성을 확보했습니다.

### 2. DHTMLX Gantt 이벤트 연동
- `onMouseMove` 이벤트를 사용하여 마우스 위치를 실시간 추적합니다.
- 마우스가 휴일이 등록된 날짜 헤더 위에 있을 때만 커스텀 툴팁을 표시하도록 구현했습니다.
- `onMouseLeave` 이벤트를 통해 간트차트 영역을 벗어날 때 툴팁을 확실히 숨기도록 처리했습니다.

### 3. 디자인 일관성 확보
- 기존 업무(Task) 툴팁과 동일한 스타일(흰색 배경, 그림자, 둥근 모서리 등)을 적용했습니다.
- 툴팁 헤더를 `[휴일/연차 정보]`로 표시하여 시인성을 높였습니다.

### 4. 성능 및 안정성 최적화
- `fixed` 포지셔닝을 사용하여 레이아웃 간섭 없이 툴팁이 최상단에 표시되도록 했습니다.
- 화면 경계 체크 로직을 추가하여 툴팁이 브라우저 너비를 벗어나지 않도록 자동 조정합니다.
- 컴포넌트 언마운트 시 툴팁 DOM 요소를 자동으로 정리(Cleanup)하도록 구현했습니다.

## 확인 방법
1. 간트차트 '일' 단위 보기 모드로 전환합니다.
2. 휴일이나 연차가 등록되어 배경색이 다른 날짜 헤더(예: 2 (월))에 마우스를 올립니다.
3. 즉시 툴팁이 나타나는지 확인하고, 디자인이 업무 툴팁과 일치하는지 확인합니다.
