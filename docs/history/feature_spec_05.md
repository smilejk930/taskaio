# Feature Spec: WBS 업무 등록 버튼 시인성 개선

## 목적
현재 '업무 목록'(WBS) 탭 하단에 위치한 '업무 등록' 버튼이 Ghost 스타일로 되어 있어 배경(Muted)과 구분이 잘 안 된다는 사용자 피드백을 반영하여, 시인성이 높은 Primary 스타일로 변경하고 디자인을 강화합니다.

## 사용자 시나리오
1. 사용자가 '업무 목록' 탭을 선택한다.
2. 하단 툴바에서 눈에 띄는 색상의 '업무 등록' 버튼을 즉시 발견한다.
3. 버튼을 클릭하면 WBS 그리드 최하단에 새로운 행이 추가되며 포커스가 이동한다.

## DB 변경사항
- 없음 (기존 스키마로 구현 가능)

## 구현 목록
### Components
- **WbsGrid.tsx**: 
  - 하단 툴바의 '업무 등록' 버튼 스타일 변경: `variant="ghost" text-primary` → `variant="default"` (또는 커스텀 Primary 색상)
  - 빈 화면(Empty State)의 '업무 등록' 버튼 스타일 변경: `variant="outline"` → `variant="default"`
  - 버튼에 호버 효과 및 약간의 애니메이션 추가 (Rich Aesthetics 준수)

### 재사용
- **shadcn/ui Button**: 기존 버튼 컴포넌트 활용

## 엣지 케이스 처리
- **그리드 데이터가 많을 경우**: 하단 툴바가 `flex-col`로 하단에 고정되어 있어 항상 접근 가능함.

## 완료 기준
- [ ] '업무 등록' 버튼의 배경색이 Primary 색상으로 변경되어 시인성이 확보됨.
- [ ] 버튼 클릭 시 정상적으로 새로운 행이 생성됨.
- [ ] 데이터가 없을 때 표시되는 중앙 버튼도 동일한 스타일로 통일됨.
