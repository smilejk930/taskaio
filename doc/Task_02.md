# 기능 추가 요청(좀더 상세히 다듬기)

## 프로젝트
- 프로젝트 정보 수정하는 화면에서 프로젝트 삭제하기 기능 추가
- 삭제기능: 소프트 삭제(삭제여부)
- 삭제 하기 위한 조건: github에서 프로젝트 삭제할때 사용하는 기법 사용
```
해당 프로젝트를 삭제할것인가요?
삭제를 원하면 해당 글자를 정확히 입력 후 삭제
예를 들어 프로젝트명
```

## 대시보드
- 고우선순위 업무 표시 안됨(데이터 확인 필요)
- 마감 임박 엄무에서 마감일을 `마감일: 2026-03-10` 이렇게 표시.(기존: 2026-03-10T00:00:00+00:00)
- 업무 클릭 시 WBS 리스트의 해당 항목으로 이동
## WBS 리스트
- 업무내용: tasks 테이블의 description 컬럼 사용안하는 문제 수정,
- 우선순위, 상태 수정할 수 있게 변경(현재는 수정못함)
- 업무 등록 시 간트뷰에서 보여줄 색상을 랜덤으로 db에 저장하고 수정할 수 있도록추가
- 업무 등록 시 매니저라면 담당자를 선택할 수 있게 변경, 일반 팀원이라면 본인이 담당자로 자동 선택
- 진척률은 10%단위로도 선택할수 있게끔 수정

## 간트뷰
- 제목 누르면 업무내용 볼수 있게 expand all 할 수 있게 수정(또는 가장 효율적인 방법으로 표현)
- 업무 추가 시 wbs 리스트에서 등록할때 사용 되는 기능 전부 사용할 수 있게 수정
- 업무 추가 시 연월일로 등록할 수 있게 변경
